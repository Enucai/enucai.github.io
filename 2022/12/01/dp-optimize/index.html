P3780 [SDOI2017]苹果树

首先原题那个 $t-h\le k$ 的条件可以转化为最多取 $t$ 个点，然后在免费取一条链（显然你会把这条链取到底部）。

然后考虑这个树形依赖不是很好处理，可能父亲节点不去完当前节点就取了。

于是把每个值不为 $1$ 的节点拆成两个节点，个数分别为 $1$ 和 $a_u-1$。把 $a_u-1$ 的节点挂在 $1$ 的节点下，这样就满足树形依赖了。

然后我们记下树的后序遍历，在后序遍历上做多重背包，再把每个节点的邻接表翻转，再在后序遍历上做多重背包。

我们发现一棵树除了一条链外，其他两个部分可以被看作第一个后序遍历上的一个前缀和第二个后序遍历上的一个前缀。

于是枚举一下免费取的那条链，把连边的背包信息合并起来取个 $\max$ 即可。

多重背包需要单调队列优化，复杂度 $O(nk)$。

P1399 [NOI2013] 快餐店

朴素做法是枚举割掉环上的哪条边，然后把剩下的树的直径求出来，然后所有的直径取个 $\min$，再除二即可。

因此我们就要快速算出割掉环上的每条边后树的直径。

直径有两种可能，第一种是不跨环的，第二种是跨环的。不跨换的可以直接算出。

而跨换的可以预处理出每个子树的深度，然后直接破环成链。枚举完边后，直径可能都在左边，都在右边，或左右各一个。

因此只要预处理出每个前缀取两个子树的距离 $\max$，每个后缀取两个子树的距离 $\max$，与每个前后缀取一个点的 $\max$。

复杂度线性就做完了。

CF372C Watching Fireworks is Fun

单调队列优化 dp 板板题，每相邻两次烟火表演之间正着倒着做两遍 dp 就好了。

$f_{i,j}$ 表示第 $i$ 次表演后，在位置 $j$ 损失的最少喜悦值。

大于小于号写错，调了好久。

CF1428G2 Lucky Numbers (Hard Version)

神仙题，琢磨了 1h 周康阳的 sol，终于把他解释清楚了。

首先有结论每个数 $n$ 都可以被拆成一堆只有 $0,3,6,9$ 构成的数与至多一个含有其他数字的数。

证明考虑按位，如果这一位有一堆 $0,3,6,9$ 与两个其他数字 $a,b$，如果 $a+b\le9$ 那么就能变成 $0$ 与 $a+b$，否则变成 $9$ 与 $a+b-9$。

于是我们先考虑每个数只被分成一个数时，他的贡献，这个计算是平凡的。

然后我们考虑加一堆由 $0,3,6,9$ 构成的数。考虑背包。

对于一个十进制位 $i$，我们可以把他拆成 $3(k-1)$ 个体积为 $3\times 10^i$，价值为 $F_i$ 的物品。

这样进行背包一定是合法的，于是本题就做完了。

但是数据范围较大，不能直接背包。这是一个多重背包，可以考虑二进制分组优化。

P4093 [HEOI2016/TJOI2016]序列

很容易想到本题就是求一个最长三位偏序子序列长度，那么直接树套树优化一下 dp 即可。

但是这样很难写，考虑 cdq 分治，这样可以在相同时间复杂度内完成该问题，$O(n\log^2n)$。

同样的 cdq 分治优化 dp 的题目还有P2487 [SDOI2011]拦截导弹。

CF739E Gosha is hunting

首先考虑每个神奇宝贝不可能同时有两个及以上的宝贝球去抽，也不可能同时有两个及以上超级球去抽。

因为抽一个多次还不如抽另一个，这样增加的期望更多。

于是我们令 $f_{i,j,k}$ 表示处理了前 $i$ 个超级宝贝，用了 $j$ 个宝贝球，$k$ 个超级球时的最大期望个数。

那么转移很容易：

分别表示啥都不选，选一个宝贝球，选一个超级球，选一个宝贝球和一个超级球。

然后发现对于固定的 $i,j$，$f_{i,j,k}$ 是关于 $k$ 的凸函数，因此考虑 wqs 二分。

通过这道题终于明白了 wqs 二分二分的其实是去切 dp 值凸函数的那个一次函数的斜率。

理解之后就是 wqs 二分板板题了。

子序列变成了子段，简单了不少。

不难发现一个区间合法，当且仅当对于任意位置 $i$，都有 $Li\le\min\limits{j\ge i}R_j$。

然后我们想求出对于每个 $r$ 的最左边能到达的 $l$，发现这个东西是由单调性的。

于是套上决策单调性模板即可，再写一个 ST 表。

但是这题可以用单调栈做到线性，想法也非常简单。
